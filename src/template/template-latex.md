* Inline equations are replaced with `\\(` *text* `\\)`

```
    \\( \mathbb{N} = \{ a \in \mathbb{Z} : a > 0 \} \\)
```

\\(\mathbb{N} = \{ a \in \mathbb{Z} : a > 0 \}\\)

```
    \\[ \mathbb{N} = \{ a \in \mathbb{Z} : a > 0 \} \\]
```

\\[\mathbb{N} = \{ a \in \mathbb{Z} : a > 0 \}\\]
